Sidenav.js

No dependencies! Only 2K when minified and gzipped!

Sidenav.js is a JavaScript library that enables users add sidebars to their web site that behaves like a native side navigation component of a mobile phone application.

This library tries to emulate the exact same behavior of Android's sidenav in pure JavaScript to enable users to create native-like web applications. Open Google Play Store and swipe your finger from the left of the screen. This library tries to provide the exact same behaviour.
